# HADAS: Tool for Analysis and Development of Sustainable Applications

This repository contains the replication package for the paper **"Finding Correlations of Features Affecting Energy Consumption and Performance of Web Servers Using the HADAS Eco-Assistant"**, currently **Published** at *SPRINGER Computing* journal.

This serves as supplementary material to support the reproducibility of the experiments and case studies presented in the manuscript.
